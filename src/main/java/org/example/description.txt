Приложение 1
Парсинг(прописать несколько видов проверок, для разного формирования файла отчетов)
    Проверки:
    1. Проверка суммы перевода(отрицательная сумма перевода)
    2. Нулевая сумма перевода или поле суммы не заполнено
    3. Некорректно указан счет получателя/отправителя(невадлидный номер)
    4. Кол-во денежных средств на счете отправителя меньше суммы перевода

    2. Применение счётчика к формированию названия файлов
